# react-native-smart-gplaces

A highly optimized, dependency-free, customizable React Native library for the **Google Places API**.

## Installation


```sh
npm install react-native-smart-gplaces
```


## Usage


```js
import { multiply } from 'react-native-smart-gplaces';

// ...

const result = await multiply(3, 7);
```
